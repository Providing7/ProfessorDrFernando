# **App Name**: Perfil Profissional

## Core Features:

- Seção Sobre: Apresentar uma biografia detalhada do professor, destacando sua experiência, missão e abordagem profissional.
- Informações de Contato: Exibir informações de contato como localização, email e telefone para facilitar o contato.
- Links de Mídias Sociais: Incluir links para perfis profissionais como LinkedIn e website para expandir a presença online do professor.
- Seção de Certificações: Exibir as certificações do professor, validando suas qualificações e especializações.
- Seção de Palestras: Apresentar uma lista de palestras relevantes ministradas pelo professor, destacando sua experiência como orador.
- Estatísticas e Reconhecimentos: Apresentar informações quantitativas, como número de profissionais capacitados, anos de experiência e nível de satisfação, para reforçar a credibilidade do professor.

## Style Guidelines:

- Cor primária: Azul claro (#A8D0E6), evocando profissionalismo e confiança.
- Cor de fundo: Branco acinzentado suave (#F5F5F5), para um visual limpo e moderno.
- Cor de destaque: Azul mais escuro (#4A6E8A), utilizada para títulos e links.
- Fonte para títulos: 'Poppins', sans-serif, para um visual contemporâneo e preciso.
- Fonte para o corpo do texto: 'PT Sans', sans-serif, garantindo legibilidade e conforto na leitura.
- Utilizar ícones claros e minimalistas para representar informações de contato e estatísticas.
- Layout limpo e organizado com seções bem definidas para facilitar a navegação e a leitura.
